## [1.1.0] - 2026-04-27

### Added

- CI pipeline (GitHub Actions).
- `ParamWriteAnalyzer` interprocedural read-only pointer/reference analysis in `TaskGraph`.
- Python benchmark harness with `readOnlyParam`, `dotProduct`, and `callChain`.
- Known-failure support in transformation test runners.
- 23 targeted test cases across 5 transformations:
  - `MultipleDeclSplitter`: `constMultiDecl`, `volatileMultiDecl`
  - `DeclarationSplitter`: `nestedScopeDecl`, `constexprDecl`, `autoDecl`
  - `GotoTransfo`: `whileEarlyReturn`, `switchReturn`, `ternaryReturn`, `multipleExitLabels`, `lambdaReturn`, `returnInDoWhile`
  - `TaskGraph`: `globalVariable`, `readOnlyParam`, `multipleReturnDep`, `independentCalls`, `ternaryDep`, `structFieldDep`, `whileLoopBody`, `forLoopBody`, `nestedCallDep`, `multipleWritesSameVar`, `outputParam`
  - `ConditionUnstack`: `nestedCondDecl`

### Changed

- Fork project on GitHub.
- Improved `TaskGraph` dependency classification for read-only non-const pointer/reference parameters.
- Updated expected outputs and dependency graphs after test-suite audit.
- Improved repository structure, CI setup, and documentation for build/test/reproduction workflows.

### Fixed

- Fixed `GotoTransfo` return rewriting under unbraced control statements (`multipleExitLabels`).
- Fixed parasitic statements generated by `DeclarationSplitter` for declarations without initializers.
- Fixed `Constify` consistency between method definitions and forward declarations.
- Fixed missing/outdated expected files and fragile oracle mismatches.
- Fixed non-deterministic generated outputs affecting test stability.

### Validation

- Test status improved from 74 executed / 65 validated cases to 97 executed / 87 validated cases.
- Transformation suites improved from 7/11 passing to 11/11 passing under the known-failure policy.
